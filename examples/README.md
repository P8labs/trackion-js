# SDK Examples

This directory contains usage examples for all supported SDK variants.

## Variants

- `web/`: Vanilla JS/TS module usage with `@trackion/js`
- `react/`: React hooks/provider usage with `@trackion/js/react`
- `vue/`: Vue 3 composables usage with `@trackion/js/vue`
- `node/`: Server-side usage with `@trackion/js/node`
- `script/`: Script-tag usage with hosted `t.js`

## Notes

- Replace `PROJECT_API_KEY`, `PROJECT_UUID`, and server URL placeholders before use.
- Examples focus on integration patterns, not full production app setup.
